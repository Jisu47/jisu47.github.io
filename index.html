활용 방법

소개 한 줄 소개와 현재 관심사, 사용하는 기술을 적어두면 시작 페이지가 됩니다.
프로젝트 GitHub 저장소, 데모 링크, 짧은 설명만 넣어도 포트폴리오처럼 보입니다.
연결 이메일, 블로그, SNS를 추가해 방문자가 바로 연락할 수 있게 만들 수 있습니다.

다음 수정

  • 문장 바꾸기 현재 소개 문장을 본인 소개와 활동 내용으로 바꾸면 페이지 성격이 분명해집니다.
  • 프로젝트 추가 대표 저장소 2~3개만 골라 제목, 설명, 링크를 넣는 것이 가장 효과적입니다.
  • 연락처 연결 GitHub 외에 메일이나 블로그 주소를 추가하면 방문 동선이 훨씬 좋아집니다.
